## About SG Window

Description: SG Window is a multipurpose flexible WordPress theme with responsive design and configurable front page. Theme equipped by one page navigation. It has many options in the theme customizer (per page sidebars, layouts: right column, left column, two columns, full width; blog: from 1 to 4 columns; favicon; logo; all google fonts; 2 color schemes, 3 nav menus and much more).

SG Window has 4 footers, 4 per page sidebars and 10 custom widgets. It can be used for portfolio, blog, e-commerce, business, landing page websites.

Supports popular plugins: WooCommerce and portfolio from Jetpack. You can find the demo at http://wpblogs.info/demo/sg-window/

## How to install

	1. Unzip and then copy the sg-window directory to /wp-content/themes/ in your WordPress installation. 
	2. Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Widgets can be found at "Appearance >> Widgets"
